## 1. 서비스 약관 요약

### 1.1 서비스 개요

AQTS(AI Quant Trading System)는 인공지능 기반 자동매매 보조 시스템입니다. 본 시스템은 시장 데이터 분석, 투자 신호 생성, 포트폴리오 구성 및 주문 집행을 자동으로 수행합니다.

### 1.2 서비스 유형

| 유형 | 설명 | 사용자 개입 |
|------|------|------------|
| ADVISORY | AI가 투자 의견을 제시하고 사용자가 최종 결정 | 주문 전 사용자 확인 필수 |
| DISCRETIONARY | AI가 사전 설정된 규칙 내에서 자동 매매 실행 | 사전 파라미터 설정 후 자동 |

---

## 2. 투자 위험 고지 (면책 고지)

### 2.1 일반 면책 조항

본 시스템을 통한 투자에는 원금 손실을 포함한 상당한 위험이 수반됩니다. 과거 수익률은 미래 수익을 보장하지 않으며, AI 분석 결과는 투자 권유가 아닌 참고 정보입니다.

### 2.2 구체적 위험 요인

**시장 위험**: 주식·환율·금리 변동에 따른 투자 손실이 발생할 수 있습니다. 레짐 전환(급등·급락·횡보·고변동) 시 기존 전략의 성과가 크게 달라질 수 있습니다.

**시스템 위험**: 서버 장애, 네트워크 지연, API 오류 등 기술적 문제로 인해 주문이 지연·실패·중복 집행될 수 있습니다. 서킷 브레이커가 동작하더라도 모든 손실을 방지하지는 못합니다.

**AI 모델 위험**: AI 분석은 학습 데이터와 모델 가정에 의존하며, 예측 불가능한 시장 상황에서는 부정확할 수 있습니다. 모델 드리프트가 감지되더라도 즉시 대응하지 못할 수 있습니다.

**유동성 위험**: 거래량이 적은 종목의 경우 원하는 가격에 체결되지 않거나, 슬리피지가 예상보다 클 수 있습니다.

**환율 위험**: 해외 자산 투자 시 환율 변동으로 원화 기준 손익이 달라질 수 있습니다.

### 2.3 손실 한도 안내

| 보호 장치 | 기준 | 동작 |
|----------|------|------|
| 일일 손실 한도 | 포트폴리오 -3% | 당일 매매 자동 중단 |
| 주간 손실 한도 | 포트폴리오 -5% | 매매 중단 + 운영자 수동 승인 재개 |
| 비상 리밸런싱 | 동적 임계값 (변동성 연동) | 방어 포트폴리오 전환 |
| 킬 스위치 | 수동 발동 | 모든 매매 즉시 중단 |

위 보호 장치는 손실을 최소화하기 위한 것이나, 시장 급변 시 정확한 수준에서의 손절을 보장하지 않습니다.

## 3. 데이터 처리 안내

### 3.1 수집 데이터

- 투자자 프로필 (위험성향, 투자 스타일, 손실 허용도)
- 주문 및 체결 기록
- 포트폴리오 보유 현황
- 시스템 사용 로그

### 3.2 데이터 보호

- 개인정보 7종 자동 마스킹 (주민번호, 전화번호, 이메일, 계좌번호, 카드번호, IP, API키)
- 감사 로그 SHA-256 해시 체인으로 무결성 보장
- 거래 기록 5년, 감사 로그 10년 보존
- 비밀키 자동 로테이션 관리

### 3.3 데이터 삭제 요청

관련 법령이 허용하는 범위 내에서 개인 데이터 삭제를 요청할 수 있습니다. 다만 법적 의무 보존 기간이 적용되는 거래 기록 및 감사 로그는 보존 기간 만료 후 삭제됩니다.
